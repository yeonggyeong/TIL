# 00_Python Basic

## 변수
- 변수는 컴퓨터 메모리 어딘가에 저장되어 있는 객체를 참조하기 위해 사용되는 이름
- 문자, 정수, 실수 등 여러 가지의 type이 존재

### 변수 선언 및 type 확인
- 변수는 `=`으로 할당
- 변수의 타입을 확인하기 위해서는 `type()`를 사용

```python
# 1) 변수 선언
x = 'GITHUB'
# 2) 변수 타입 확인
type(x)
# 3-1) 동시 같은 값 할당
x = y = 100 # x = 100, y = 100 으로 값 할당
# 3-2) 동시 여러 값 할당
x, y = 1, 2 # x = 1, y = 2 으로 값 할당
 ```

2)의 결과는 x에 저장된 값에 따라 확인 가능
